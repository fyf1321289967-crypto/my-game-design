# 增长
单局累计，单局结束清零。
随捡起[[enemy/normalenemy\|normalenemy]]掉落的金币增长，每捡起1金币，[[machanism/greed\|greed]]增加1。
当游戏结束（无论通没通关）如果[[machanism/greed\|greed]]超过100，失去所有单局获得的[[material/token\|token]][[material/blood token\|blood token]][[material/soul token\|soul token]]，解锁[[machanism/greedy\|greedy]]
[[machanism/greed\|greed]]超过500直接暴毙，失去当前单局获取的所有[[material/token\|token]]。
