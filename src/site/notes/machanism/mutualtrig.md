# 机制
当某一牌型开启了mutualtrig，当牌组包含它就可以触发这类攻击。
例如高牌开启mutualtrig，则无论打出什么牌型都可以触发这类攻击。
例如在游戏内所有牌型都开启了mutualtrig，则打出五条时可同时发出四条葫芦三条两对对子高牌的攻击。
鉴于顺子同花同花顺无法触发mutualtrig，这些牌型开启mutualtrig后，判定的五张只要有完全相同的牌便视为需要的牌
例如：所有牌型都开启了mutualtrig
spade 2 spade 2 spade 3 spade 3 spade 3
可以视为spade2-6的同花顺，同时也可判断为除五条和四条外的所有牌型
# 开启方式
当凑齐三个牌型触发器碎片可以自选开启一类牌型的mutualtrig
